# Amazon Clone – Frontend Project

## Overview

This project is a structured frontend clone of the Amazon homepage built using HTML5 and CSS3. The objective was to replicate the layout, styling, and visual structure of a real-world e-commerce platform to strengthen core frontend development skills.

---

## Key Highlights

* Clean and structured HTML architecture
* Responsive navigation bar layout
* Product display sections
* Styled search bar and banner area
* Footer section aligned with modern UI standards
* Organized file structure for scalability

---

## Tech Stack

* HTML
* CSS
* Flexbox

---

## What I Learned

* Building complex page layouts using pure HTML and CSS
* Understanding spacing, alignment, and positioning
* Implementing Flexbox for responsive alignment
* Writing clean and readable frontend code
* Structuring projects in a professional format
